# gcal-tasks-deployer

効率的なスケジュール管理のために設計されたブラウザ拡張機能です。スクリーンショットを読み込み、AIによる分析を経て、画像内の予定情報をワンクリックでGoogleカレンダーおよびGoogle Tasksに素早く同期します。

### 特徴
- **スクショ自動認識**：スクリーンショット内の日時、イベント内容などの重要情報を自動で抽出。
- **ワンクリック登録**：手動入力の手間を省き、スケジュールを即座に追加。
- **マルチAIモデル対応**：**Gemini**、**OpenAI**、および **DeepSeek** のAPIに対応。

### インストール方法
1. 本リポジトリのコードをローカルにダウンロードします。
2. Chromeで `chrome://extensions/` を開きます。
3. 右上の **「デベロッパー モード」** をオンにします。
4. ダウンロードしたフォルダを画面にドラッグ＆ドロップします。

### 注意事項（事前準備）
- 本拡張機能には、APIキーは含まれていません。
- **ご利用前にご自身で設定してください**：AIのAPIキー、およびGoogleの資格情報（`Google Client ID` と `Google Client Secret`）。
- **プライバシー保護**：設定されたAPIキーなどのデータは、すべてブラウザ内（ローカル）にのみ保存され、外部に送信されることはありません。
