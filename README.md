# gcal-tasks-deployer

A powerful browser extension designed for efficient schedule management. By capturing screenshots and utilizing AI for content analysis, it helps you synchronize event information directly to your Google Calendar and Google Tasks with a single click.

### Features
- **Smart Screenshot Recognition**: Automatically extracts key information such as dates, times, and event details.
- **One-Click Deployment**: Eliminates manual entry by automatically formatting and adding tasks.
- **Multi-Model Support**: Supports API configurations for **Gemini**, **OpenAI**, and **DeepSeek**.

### Installation
1. Download all the code files of this repository to a local folder.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Turn on the **"Developer mode"** toggle in the top-right corner.
4. Drag and drop the downloaded folder directly into the Extensions page.

### Prerequisites & Configuration
- This extension does not come with built-in API keys.
- **Before use, you must configure**: AI platform API Keys, and Google Credentials (`Google Client ID` and `Google Client Secret`).
- **Privacy Notice**: All sensitive data are stored strictly within your local browser storage and will never be uploaded externally.
